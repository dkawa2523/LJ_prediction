# Glossary（用語集）

- **Dataset**：入力CSV/SDFから構築された学習用データ集合
- **Task**：目的変数と損失/評価指標の定義（回帰/分類/マルチタスク）
- **Preprocess**：分子の正規化、欠損処理、3D生成など
- **Featurizer**：FP、RDKit記述子、3D幾何などを特徴量化する処理
- **Model**：予測器（FP+ML、GNN、Transformer、事前学習モデルなど）
- **Trainer**：学習ループ（最適化、early stopping、ログ）
- **Artifact**：学習・推論の成果物（モデル、指標、予測、設定）
- **Skew**：学習と推論で前処理/特徴量がズレる問題
- **ADR**：Architecture Decision Record（設計決定の記録）
- **RFC**：仕様変更提案（ADR前段）
